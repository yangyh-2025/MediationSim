## 9. 已修复的缺陷

| 编号 | 发现途径 | 问题描述 | 修复措施 |
|------|----------|----------|----------|
| BUG-1 | E2E | `bootstrap_mediation()` 返回缺少 `bootstrap_samples` 字段 | 添加 `bootstrap_samples` 键 |
| BUG-2 | E2E | 路由收集直接覆盖，未合并同一路径的多方法 | 改为集合并集 `\|` |
| BUG-3 | 集成 | `evaluations/trigger` 路由 `except Exception` 吞噬 `HTTPException` | 添加 `except HTTPException: raise` |
| BUG-4 | 集成 | `statistics/run` 路由同上问题 | 同样添加 HTTPException 保护 |
| BUG-5 | 集成 | H4 test ANOVA 在稀疏分类数据上崩溃 | 添加 try/except 回退逻辑 |
| BUG-6 | 集成 | NaN 值写入 SQLite 导致 `NOT NULL` 约束失败 | `save_analysis_result` NaN 安全处理 |
| BUG-7 | 集成 | Pydantic `HypothesisResult` JSON 序列化 NaN 失败 | 添加 `@field_serializer` NaN→0.0 |
| BUG-8 | Ruff | 未使用 import + f-string 无效前缀 | 清理 import，去除 `f"..."` |
| BUG-9 | Ruff | 单行超 120 字符 | 拆分长字符串 |
| BUG-10 | 配置 | 端口 8000/5173 与系统常用端口冲突 | 改为 59870/59871 |
| BUG-11 | 契约 | `ExperimentConfigIn.name` 无 min_length 约束 | 添加 `min_length=1` 和 `max_length=200` |
| BUG-12 | 契约 | `conditions` 字段无合法值校验 | 添加 `@field_validator` |
