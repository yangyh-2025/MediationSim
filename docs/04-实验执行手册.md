## 实验流程

### Phase 1：预实验（35 次）

1. 前端 → 实验配置页 → 创建新实验
   - 名称：`预实验校准`
   - 条件：全选 7 条件
   - 每条件运行：5 次
   - 启动实验
2. 监控页观察智能体行为是否符合预期
3. 查看评估报告，确认参数校准

### Phase 2：主实验（210 次）

1. 创建新实验 `主实验`
2. 每条件 30 次，全选 7 条件
3. 启动后系统自动执行，每 10 次触发快速评估迭代
4. 监控页可实时查看进度和协议率

### Phase 3：中介效应检验

1. 创建新实验 `禁用附带支付`
2. side_payment_enabled = false
3. 仅选亲强条件（H-PS, L-PS, CD）
4. 启动后与主实验数据合并进行 Bootstrap 中介检验

### Phase 4：统计分析

1. 主实验完成后 → 结果总览页
2. 点击"运行统计分析"按钮
3. 系统自动执行：
   - H1 独立 t 检验 + Logistic 回归
   - H2 ANOVA + Kaplan-Meier + Cox PH
   - H3 Bootstrap 中介检验
   - H4 双因素方差分析

## 关键参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| temperature | 0.7 | LLM 温度，越高多样性越大 |
| max_tokens | 3840 | 每轮最大输出 tokens |
| max_rounds | 10 | 单次模拟最大谈判轮数 |
| runs_per_condition | 30 | 每实验条件运行次数 |
| side_payment_budget_pct | 0.5 | 附带支付预算占 GDP 百分比 |
